# Code of Conduct

This project adopts the [Contributor Covenant][cc], version 2.1, as its code of
conduct. The full text is available at:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

In short: participants are expected to be respectful, welcoming, and
constructive in all project spaces (issues, pull requests, and discussions).

## Reporting

To report a concern, use the repository's **Security** tab → **Report a
vulnerability** for a private channel, or contact the maintainer
[@takahira](https://github.com/takahira) via GitHub. Reports will be reviewed
and handled as confidentially as is practical.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
